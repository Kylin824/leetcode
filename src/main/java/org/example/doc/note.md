#### 位运算
- 判断是否为2的n次方：n & (n - 1) == 0 
- 异或（不同为1）：1 ^ 0 == 1; 1 ^ 1 == 0
- 整数取最右一位：x1 = x & 1;
- 计算2的n次方：res = 1 << n; // 把1左移n位

#### 栈操作

#### 队列操作
- Queue<T> queue = new LinkedList<>();

- x = queue.poll();
