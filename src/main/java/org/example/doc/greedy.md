## 一、区间调度类（最经典）

这类题的核心是**按终点排序**，每次选择结束最早或最不冲突的区间。

| 题号 | 题目 | 贪心策略 | 难度 |
|---|---|---|---|
| **435** | 无重叠区间 | 按结束时间排序，保留最多不重叠区间，移除最少 | 🟡 中等 |
| **452** | 用最少数量的箭引爆气球 | 按结束位置排序，每次射穿最多重叠区间 | 🟡 中等 |
| **56** | 合并区间 | 按起点排序，能合并就合并（不算严格贪心，但常一起练） | 🟡 中等 |
| **986** | 区间列表的交集 | 双指针，每次取交集后移动结束较早的指针 | 🟡 中等 |
| **646** | 最长数对链 | 按数对第二位升序排序，选尽可能多的链 | 🟡 中等 |

## 二、分配/调度类（“给”与“取”）

核心是**排序后匹配最小可行资源**或**优先级调度**。

| 题号 | 题目 | 贪心策略 | 难度 |
|---|---|---|---|
| **455** | 分发饼干 | 胃口和饼干都排序，最小饼干喂最小胃口 | 🟢 简单 |
| **406** | 根据身高重建队列 | 按身高降序、k升序排序，按k插入 | 🟡 中等 |
| **621** | 任务调度器 | 优先执行剩余次数最多的任务（或公式计算空闲时间） | 🟡 中等 |
| **135** | 分发糖果 | 两次遍历：从左到右、从右到左，满足相邻规则 | 🔴 困难 |
| **881** | 救生艇 | 排序后双指针，最轻配最重 | 🟡 中等 |

| 题号 | 题目 | 贪心策略 | 难度 |
|---|---|---|---|
| **55** | 跳跃游戏 | 维护当前能跳到的最远位置，能覆盖终点即可 | 🟡 中等 |
| **45** | 跳跃游戏 II | 每一步在可达范围内跳到能扩展最远位置的点 | 🟡 中等 |
| **134** | 加油站 | 总油量>=总消耗时，从累计最低点之后出发 | 🟡 中等 |

## 九、学习建议

1. **先做简单题**（455、860、605），建立贪心的直觉——“先排序再决定”。
2. **区间题集中刷**（435、452、646），练熟“按 end 排序”的套路。
3. **跳跃题对比刷**（55 和 45），理解“可达性”和“最小步数”的区别。
4. **每道题做完后追问自己**：
    - 为什么贪心选择是安全的？（即为什么局部最优不损害全局最优）
    - 如果改一下输入，贪心还会成立吗？（找反例训练判断力）
